Grupparabete Webbutveckling 2
Grupp 2: Christer och Johan

Uppgift:
Skapa en iteraktiv webbplats med hjälp av HTML, CSS och JavaScript.

Krav:
- En responsiv design som anpassar sig tiil olika skärmstorlekar.
- Ett formulär som användare kan använda för att skicka in iformation.
- Implementera Git för att versionera och spåra ändringar i din kod.
- Använd Git-Hub för att dela din webbplats och samarbeta med andra.

Tips:
- Börja med att skissa en wireframe för din webbplats för att planera layouten och funktionaliteten.
- Dela upp projekt i mendre delmål och arbeta metodiskt.
-Använd online-verktyg och resurser för att lära dig mer om HTML, CSS och JavaScript.
- Var kreativ och ha roligt.

Bonus:
Du kan använda Jquery; Angular och React i programmet för att förenkla och effektisera utvecklingen av interaktiva och dynamiska webbplatser.

Bedömning:
Din Webbplats kommer att bedömas utifrån följande kriterier.

- Funtionell korrekthet
- Användarvänlighet
- Kvaliteten på din kod
- Kreativitet

- Använd relevanta bilder och video för att förbättra användarupplevelsen.
- Optimera din webbplats för sökmotorer (SEO).

- Komprimera ditt arbete (Mappar & filer) och bifoga den via Campus.
- Se till att ZIP-filnamnet inkludderar ditt och din arbetskamrats namn.
